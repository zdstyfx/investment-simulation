# 07 · 界面与交互规格

整体风格:**暗色交易终端**(类华尔街盯盘)+ 8-bit 像素科技感 + 韶音运动科技调性。仅桌面端。
"爽"来自极紧的反馈回路:每个操作都立刻有回应,但不拖慢操作。大动效集中在大屏揭榜。

## 1. 设计令牌

| 用途 | 值 |
|---|---|
| 页面底 | `#0A0D12`(大屏可更深 `#070A0F`) |
| 面板 | `#121823`,边框 `rgba(255,255,255,.06~.1)`,圆角 12–16px |
| 主文字 / 次文字 | `#E7ECF3` / `#8A93A3` |
| 主强调色(可用资金/在线/品牌) | 电光青 `#2DD4D9` |
| 正向/确认 | 绿 `#39E08C`;负向用红 |
| 像素字体 | `Press Start 2P`(Google Fonts),用于大额数字、榜单标题/数值、名次 |
| 正文字体 | 系统 sans |
| 等宽数字 | `font-variant-numeric: tabular-nums` |

**每组固定颜色**:按 `seat_no` 从一套高区分度色板派生(如青/紫/蓝/琥珀/粉/绿/珊瑚…),全系统一致(方块、图例、持仓圆点、榜单)。当前路演组的颜色也用于小组端的投资按钮/滑块强调色。

动效一律包在 `@media (prefers-reduced-motion: no-preference)` 内。

## 2. 小组端 `/play`(电脑浏览器)

核心是一块"持仓地图 + 像素余额"。

- **顶栏**:活动名 + 场次/组号;右侧"投资进行中"状态(脉冲绿点)+ 本轮倒计时(mm:ss)。
- **可用资金(招牌组件)**:
  - 大号**像素字**金额(`Press Start 2P`),带青色辉光 `text-shadow`;数值变动时辉光增强并缓动滚动到新值。
  - **资金方块格子(GitHub 贡献图风格)**:共 `initial_funds / invest_step` 个小方块(默认 100 个,每格 ¥10,000)。
    - **可用** = 实心亮青 `#2DD4D9` + 轻微辉光。
    - **已投出** = 不消失,而是变成**对应目标组颜色的半透明残影(约 50% 不透明度)**,留在格子里。
    - 投资时,边界处的"亮青"方块**平滑染色**成目标组的幽灵色(`background`/`opacity` 过渡 ~0.4s)。整张格子即"这局资金分配地图"。
    - 布局:已投出方块(按组分段、各组一色)在左,剩余亮青在右;或等价镜像。配一行图例:`■ 可用 X万 · ■ 第5组 15万 · ■ 第1组 8万 …`。
- **投资控制**:滑块(step=invest_step,max=当前余额)+ 快捷按钮(10% / 25% / 50% / ALL IN / 清空)+ "确认投资"主按钮(按钮配色 = 当前路演组颜色)。确认后:余额滚动+辉光、方块染色、右上角回执 toast 滑入、按钮闪一圈、持仓新增一行。
- **正在路演卡**:当前组号(带该组颜色圆点)+ 产品名 + 方向标签 + "已路演 k/N 组"。
- **我的投资组合**:列表,每行带该组颜色圆点 + 组号 + 产品 + 金额。**只显示自己的**。

## 3. 主办方控制台 `/admin`

两块:**新建/配置场次** 与 **现场操盘**。

- **新建场次配置面板**:把 `06-config-parameters.md` 的参数做成可填表单(文本框/步进器/开关/下拉/分段/标签)。"评审与结算"区块重点突出,下方实时预览结算规则:
  `融资前 N 名:由评审给 min x–max x 加成 · 第 N+1 名起:按 base x 结算`,随调随变。
- **现场操盘**(running):必须傻瓜化、抗压。一个清晰的当前状态(阶段/当前路演组/窗口开关),大按钮:设当前路演组、开窗/关窗(可带时长)、下一组、录评审倍率(任意时间)、一键结算、揭榜。展示融资/投资聚合(仅主办方可见)。

## 4. 大屏 `/screen`(免登录,投影)

现场最炸、最适合拍照的高光页;像街机记分牌。

- **融资榜揭晓**:各组竖向柱子从 0 **往上窜**,从垫底一路 stagger 到冠军(每根间隔 ~0.2s);柱顶数值**滚动计数**(可用"万"显示);前 N 名高亮 + 奖牌 + 名次(像素字),冠军柱辉光脉冲 + 皇冠落下。提供"重新揭晓"按钮供主持人重播。
- **投资收益榜**:同款形式展示 ROI / 净收益(若 reveal_show_roi)。
- **(可选)中间一屏**:评审倍率亮牌 →「融资额 × 倍率 = 最终估值」换算动画,放大乘法爽点。
- 仅在后端 `revealed=1` 后拉取榜单数据;此前不展示任何金额。

## 5. 已验证的交互(已做过可点原型,按此重建)

- 小组端:像素余额 + 方块持仓地图 + 染色投资 + toast/闪动 + 倒计时。
- 主办方:场次配置表单 + 结算规则实时预览。
- 大屏:融资榜柱状揭晓 + 奖牌/名次/皇冠依次出现 + 数值计数 + 重播。

> 配色、辉光强度、像素字号、染色/揭晓动画时长都是可调参数,留出常量便于现场微调。
